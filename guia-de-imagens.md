# Guia de Imagens — A&A Studio

Este documento lista **todas as imagens** utilizadas no site, com tamanhos recomendados e descrições do que cada uma deve conter.

## Imagens do Site

| Arquivo | Seção | Tamanho | Formato | Descrição |
|---------|-------|---------|---------|-----------|
| `logo.png` | Header / Footer | 320x90 | PNG (fundo transparente) | Logotipo da A&A Studio. Versão principal com fundo transparente. |
| `favicon-32x32.png` | Head (favicon) | 32x32 | PNG | Ícone da aba do navegador — versão reduzida do logo ou ícone "A&A". |
| `favicon-16x16.png` | Head (favicon) | 16x16 | PNG | Ícone pequeno para abas e bookmarks. |
| `apple-touch-icon.png` | Head (favicon) | 180x180 | PNG | Ícone para dispositivos Apple quando o site é salvo na tela inicial. |
| `hero-bg.webp` | Hero (#hero) | 1920x1080 | WebP | Foto principal — espaço de trabalho digital premium, equipe trabalhando em design, ou imagem abstrata tecnológica que transmita inovação e profissionalismo. |
| `sobre.webp` | Quem Somos (#sobre) | 800x600 | WebP | Foto da equipe ou do ambiente de trabalho da A&A Studio. Pode ser uma imagem que transmita colaboração, profissionalismo e confiança. |

## Caminhos no Projeto

Todas as imagens devem estar na pasta:

```
dist/assets/img/
```

## Status dos Placeholders

| Item | Status |
|------|--------|
| `logo.png` | ⚠️ **Pendente** — O usuário deve salvar a logo neste caminho |
| `favicon-32x32.png` | ⚠️ **Pendente** — Criar a partir do logo ou pedir ao usuário |
| `favicon-16x16.png` | ⚠️ **Pendente** — Criar a partir do logo ou pedir ao usuário |
| `apple-touch-icon.png` | ⚠️ **Pendente** — Criar a partir do logo ou pedir ao usuário |
| `hero-bg.webp` | ⚠️ **Pendente** — O usuário deve fornecer ou aprovar uma imagem |
| `sobre.webp` | ⚠️ **Pendente** — O usuário deve fornecer ou aprovar uma imagem |
| Google Maps iframe | ⚠️ **Pendente** — Inserir o código embed do Google Maps na seção #area-atuacao |
